# Descriptive-Answer-Evaluation-System
A stand-alone tool that is used for evaluating descriptive answers based on 3 criterion : 

1)Word Length: Length of the answer/text script.  

2)Keywords used: Checking of keywords is done with a reference answer and extraction of keywords from file is done using RAKE algorithm.

3)Syntactic Analysis (Parsing): Each sentence is checked if it is grammatically right and this is done using Language Check tool.
